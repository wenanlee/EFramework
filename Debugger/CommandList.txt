PS:区分大小写
//---------------------内建命令----------------------------------
clear:清理
close:关闭控制带
profiler:切换到profiler模式
log:切换到log模式
//-----------------------外部命令------------------------------------
print xxx:打印xxx
get 301 100:获取100个苹果
remove 301 100:移除100个苹果
set mode god:设置上帝模式((sound volume)sv:0-200)(mode:god,normal) 
set netlog yes/no:网络发送日志
set savelog [path]:保存日志 //默认为App根目录log.txt
set netlogip 60100:设置发送终端ip默认为群发 
set netlogport 60100:设置网络端口默认60100
